# TaskFlow API - Codex instructions

- Stack: API de Node/Express en `server/`, Prisma con PostgreSQL. Docker Compose levanta `db` y el contenedor `api` aplica `npx prisma migrate deploy` antes de iniciar `node src/server.js`.
- Instalacion local: en `server/` corre `npm install`. Usa `server/.env` como base; ajusta secretos antes de commitear.
- Ejecucion: con la base levantada (`docker compose up -d db`), usa `npm run dev` o `npm start` dentro de `server/`.
- Prisma: cuando cambies el esquema en `server/prisma/schema.prisma`, corre `npx prisma generate`. Para nuevas migraciones usa `npx prisma migrate dev --name <nombre>` con la base activa.
- Tests: en `server/` corre `npm test`. Necesita `.env.test` con `DATABASE_URL` apuntando a una base de pruebas; Jest y Supertest ya estan configurados.
- Notas: manten los cambios dentro de `server/` salvo ajustes a Docker Compose. No comprometas secretos reales; los valores actuales son solo para desarrollo.
